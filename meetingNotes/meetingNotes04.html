1. 프로젝트 개요

본 프로젝트는 사용자 맞춤형 날씨 정보를 제공하는 웹 애플리케이션을 개발하여 데이터 기반 실시간 대시보드를 구현하는 것을 목표로 한다.
기존의 단순 예보 제공 서비스와 달리, 사용자가 설정한 지역을 기반으로 세부적이고 개인화된 기상 데이터와 위험 알림 기능을 제공한다.
또한 시각화를 통해 정보 이해도를 높이고, ‘날씨별 옷차림 추천’ 등 사용자 생활에 도움이 되는 기능도 함께 제공한다.

(추가사항) 날씨 랭킹, 라이트/다크모드 토글, 애니메이션 효과, 기상특보 API 연동 기능 포함

2. 데이터 설명

• 기상청 단기·초단기 예보 API 활용 (기온, 습도, 강수확률, 풍속 등 실시간 데이터).
• 지역별 기상 지표 수집 후 자체 랭킹 로직에 활용.
• 미세먼지·강풍 등 외부 위험 데이터 연동을 통한 위험도 알림 제공 가능.
• Google Chart 연동을 기반으로 대시보드 형태의 데이터 시각화 구성.
• 사용자가 설정한 지역에 대한 기상특보 정보를 API로 실시간 제공.

3. 화면설계 ※ 화면별 평가

※ 이미지를 클릭하면 크게 볼 수 있습니다.

3.1 로그인/회원가입 화면 – 비밀번호 입력 및 확인 기능, 비밀번호 암호화 처리

3.2 메인 대시보드 화면 – 지역 설정, 주요 기상정보 표시, Google Chart 시각화, 날씨랭킹, 기상특보 실시간 연동, 옷차림 알리미, 홈 버튼 추가, 라이트/다크모드 토글 테마 시스템 구성

4. 추가 기능목록 ※ 추가된 기능항목별 평가 시나리오

4.1 대시보드 홈 버튼 추가
- 사이드바 상단 사용자 환영 문구 옆에 메인 대시보드로 즉시 이동 가능한 홈(Home) 아이콘 버튼 배치.
- 목적: 페이지 이동(랭킹, 내 정보 등) 후 대시보드로 빠르게 복귀할 수 있도록 내비게이션 편의성 강화.

4.2 다크모드/라이트모드 테마 도입
- 헤더 우측 상단에 테마 전환 토글 버튼 배치.
- 기본 Light 테마 및 Real Black 기반의 Dark 테마 제공.
- localStorage를 통해 새로고침 후에도 사용자 테마 선호값 유지.
- 차트, 옷차림 추천 박스, 입력 폼 등 모든 UI 요소에 다크모드 대응 완료.

4.3 기상특보 정보 연동
- 기상청 기상특보 API를 메인 대시보드에 연동하여 실시간 특보 정보를 제공.
- 사용자 선택 지역에 특보가 발효된 경우 경고 메시지 표시.
- 특보 내용이 길어질 경우 ‘더보기/접기’ 기능을 적용해 화면 공간 효율화.
- 특보가 없을 경우 “현재 발효 중인 특보 없음” 메시지 출력.

※ 이미지를 클릭하면 크게 볼 수 있습니다.
